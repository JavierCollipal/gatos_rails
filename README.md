# Gatos rails

rest api de gatos con la gema devise jwt para lograr stateless auth.

# Deploy heroku
1-Respetar el procfile <br >
2-Generar una llave secreta <br >
3-Generar una instancia postgres en heroku <br >

# Deploy local

1-bundle install <br >
2-rake db:setup <br> 
3-rake db:migrate <br>
4-rails server
